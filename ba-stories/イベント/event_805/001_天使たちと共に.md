# プロローグ;天使たちと共に

**【山海経・とある陶磁器工房】**

**シュン**: みなさん、ここが山海経（せんがいきょう）一番の陶磁器職人である、猫田さんの工房ですよ～。

**シュン**: まずは猫田さんにあらためてご挨拶をして、今日は陶磁器についてたくさん学びましょうね！

*（梅花園の園児たち）「「「よろしくおねがいしまーーす！」」」*

**シュン**: 猫田さん。今日はこうして見学させてくださって、本当にありがとうございます。

**猫田**: ま、構わないさ。

**猫田**: 子どもたちは学園都市の未来。こうして儂の仕事を見ることで、梅花園の中に何か新しい夢を持つ子が生まれたのなら……それは、どの作品にも負けないくらい価値がある。

**シュン**: ふふ、そう言っていただけますとありがたいです。

**シュン**: では、さっそく見学を……

*（梅花園の園児A）猫田さーん、この粘土はなぁにー？*

**猫田**: ふむ、それは今度の展示会に出す予定の新しい陶磁器で――

*（梅花園の園児B）あは、もちもちして柔らかそ～！*

*（梅花園の園児B）ぶにっ！えへへっ！*

**猫田**: 儂が10時間かけて作った、陶磁器の原型がーーー！？

**シュン**: み、みなさん！勝手に作品に触っちゃだめですよ！

**シュン**: す、すみません、私の目が届かず……！

**猫田**: いや、大丈夫だ……こんなこともあろうかと、予備の原型があるからな。

**猫田**: それより、私も見間違いでなければ……あの窯の上に登っているのはまさか……。

*（梅花園の園児C）すごーい！たかーい！！あははっ！*

**シュン**: み、みなさん！窯の上は危ないから、登ってはいけません……！

**シュン**: シュンおねーちゃーん、泥団子投げられた～！！

*そっちが先に手榴弾投げてきたんじゃ～ん！*

**シュン**: お、お友達には泥団子も手榴弾も投げてはいけません！

*（梅花園の園児F）この「かま」で、とーじきを焼くのかな？*

*（梅花園の園児G）たぶん。でもなんか火が弱いね？*

*（梅花園の園児F）よし、じゃあ手伝ってあげよー！*

**シュン**: か、窯に白リン弾を投げてはいけません！

**猫田**: も、燃えている……窯が、私の工房が……っ！？

*（梅花園の園児H）シュンおねえちゃん、トイレ～！*

*（梅花園の園児I）つかれた～！*

**シュン**: み、みなさん、お願いですからちょっと待ってください……！！！
