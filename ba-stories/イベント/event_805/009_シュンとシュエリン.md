# 第8話;シュンとシュエリン

**【山海経・梅花園】**

*（梅花園の園児A）おねえちゃん、あの子たちまた喧嘩してるー！*

**シュン**: ほら、喧嘩はダメですよ。同じ梅花園のお友達なんですから、仲良くしましょう？

**シュン**: 何が喧嘩の原因なんですか？

*（梅花園の園児B）あっちが私のことチビって……！*

*（梅花園の園児C）その前に、私のメガネがカッコ悪いって言ったじゃん！*

**シュン**: 外見や、身に着けているもののことを悪く言ってはダメですよ。それぞれ個性と、好きなものがあるんですから。ね？

*（梅花園の園児B）……ごめんなさい。実はそのメガネ、本当は可愛いなって思って……でも、私も欲しいな、ズルいなって思って、つい……。*

*（梅花園の園児C）私こそ、ごめんなさい……私も、もっと大きくなりたいなって最近ずっと思ってて……。*

*（梅花園の園児C）シュンおねえちゃんみたいに、早く大きくなりたい……。*

*（梅花園の園児B）うん、シュンおねえちゃんって背も高いし頭もいいし、色々おしゃれだしすごいよね。*

*（梅花園の園児C）私もそう思う！ねえおねえちゃん、どうすればおねえちゃんみたいになれるの？*

**シュン**: （ふふっ……そうですよね、逆に子どもたちは早く大人になりたいと思うものですよね）

**シュン**: そうですねー……でも、もし背が高くなったとしても、良いことばかりではないんですよ？

**シュン**: 背が高いと、不便なこともたくさんあるんです。

*（梅花園の園児B）そうなの？高いところに手も届くし、屋根裏部屋とかにも簡単に登れるのに。*

**シュン**: もちろんそういう時には便利ですが、例えば高いところの柱に頭をぶつけたりすることもしょっちゅうです。

**シュン**: それに、私としてはむしろ、みなさんみたいに背が低いのも羨ましいなって思いますよ。

*（梅花園の園児B）えー、そうなんだ。じゃあ大きくならないで、子どものままの方が良いの？*

**シュン**: うーん、それは難しいですね……確かに背が高くて不便なこともありますが、必ずしも悪いことばかりではありませんし……。

*（梅花園の園児A）え、どういうこと？何かすっごく良いことがあるの？*

**シュン**: ……。

**シュン**: それは……

**シュン**: ふふっ、秘密です！大きくなれば、みなさんも自然と分かりますよ！

*（梅花園の園児A）えーっ！？*

*（梅花園の園児A）シュンおねえちゃんずるーい！*
