# 後日談その6;実行委員会編

**マリー**: ふぅ……落ち着くわね……

**ユウカ**: 1日の疲れが、この一杯で癒やされていく……

**マリー**: こんなに素敵な紅茶は初めて飲みます……不思議と落ち着く甘い香りがして、とっても美味しいです。

**マリー**: 適温で飲みやすいですし、良い香りが広がっていて……

**マリー**: 前に私が淹れたときは、こんなに豊かな香りはしなかったのですが……
さすがです、ハスミさん。

**ハスミ**: ふふ、ティーパーティーから素敵な茶葉を送っていただいたおかげです。

**ハスミ**: まだありますので、お代わりされる方はお申し付けください。

**ノア**: あら、カップが空いていますね。もう1杯いかがですか？

**ハスミ**: はい、お願いします。

**ノア**: ……正義実現委員会の副委員長は、お茶に厳しいと伺っていましたが、あの噂は本当だったのですね。

**ノア**: 人の手で、これほど精巧で洗練された味を出せるとは思いませんでした。

**ノア**: 貴重な経験をさせていただき、ありがとうございます。

**ノア**: 気に入っていただけたようで、何よりです。

**ノア**: その……生塩（うしお）ノアさん、ですよね……？

**ノア**: 何度かすれ違いはしましたが、こうして実行委員会の事務所でお会いしたことはなかったような。

**ハスミ**: ええ。このように言葉を交わすのは初めてですが、ユウカちゃんからお話は伺っています。

**ハスミ**: 私は今回の「晄輪大祭」で主に現場管理をしていたので、事務担当のハスミさんとはほとんど接点がありませんでしたね。

**ノア**: そうでしたか。ご挨拶が遅れてしまい、失礼いたしました。

**ハスミ**: ハスミさんは最後のリレー以外、一切参加されませんでしたよね？

**ノア**: 射撃やボール投げといった、ハスミさんの特技を活かせる競技も多かったと思うのですが……

**ノア**: あえて参加しなかった理由を聞いてもよろしいですか？

**ハスミ**: それは……

**ハスミ**: 実行委員の立場上、みなさんと一緒になって楽しむのは心苦しく……

**ハスミ**: 私が楽しんでいる間にも、誰かが自分の分まで仕事をしていると考えたら……
そのような思いで、今回は仕事に集中しておりました。

*……結果的に、最後は私も楽しんでしまいましたが。*

**ハスミ**: 今回の開催期間中、そのようなお話を何度か耳にしました。

**ハスミ**: 自分が楽しんでいる裏で、誰かが苦労している。
逆に自分が苦労しているから、誰かが楽しめている、といった内容を。

**ハスミ**: みんなが楽しめるお祭りなんて、虚構に過ぎないと。

**ハスミ**: ですが、私は自分の行動次第で何だって叶えられると思っています。

**ノア**: 合理的ではないと、ユウカちゃんからよく言われますが……
私はこの不合理な考えが結構好きなんです。

**ハスミ**: ……。

*ハスミさんはどうでしょう？
この不合理なお祭りを楽しんでいただけましたか？*

**ノア**: はい、それはもちろん。
とても楽しいお祭りでした。

**ハスミ**: ふふ。そう言っていただけると、頑張って準備した甲斐があります。

**ノア**: このように、学園外の方とお茶をして1日を終えるのも良いものですね。

**ハスミ**: 理由がなくとも、こういった場を今後も設けたいものです。

**ハスミ**: ところでハスミさん、こんな言葉を聞いたことはありますか？

**ノア**: ……帰るまでが、晄輪大祭だと。

**ハスミ**: ……はい？それは、どういう……

**ノア**: 先ほども申し上げましたが……
私は自分の行動次第で何だって実現できると思っています。

**ノア**: 気を抜かずにいきましょう。元気の有り余っている生徒たちが、「後夜祭」で何を起こすかまだ分かりませんからね。

**ノア**: 今日の出来事を振り返ると、ノアさんが懸念される気持ちも理解できますが……

**ノア**: さすがに各学園の生徒会役員が集結している日に、わざわざ騒ぎを起こすような方は――

**正義実現委員会部員**: ハスミさん、大変です！

**正義実現委員会部員**: ゲヘナの生徒が「後夜祭」のキャンプファイヤーでバーベキューをすると言い出し、警備員と衝突しています！

**正義実現委員会部員**: また、花火の色に不満を抱いた生徒が、たき火に金属粉を投げ入れようとしており……

**ハスミ**: ……。

**ノア**: あら、それは一大事ですね。

**ハスミ**: 正義実現委員のみなさんを今すぐ呼んでください。

**ハスミ**: それと、アスレチックスタジアム内にある消火設備を集めて、消火準備をお願いします。

**正義実現委員会部員**: ……ゲヘナの行政官はどちらに？

**ハスミ**: それが……アコ行政官は、ヒナ委員長を探すと言ったきり戻ってこず……

**ハスミ**: 場内の消火設備は、ゲヘナの生徒が「温泉開発に使う」と訳の分からないことを言いながら、全て持って行ってしまいました……

**正義実現委員会部員**: ……。

*ねえ、ちょっと！
ゲヘナの給食室から謎の生命体が脱走したって……！*

*……えっ！？パンケーキが空を飛んでない！？*

*絶対に……*

**ハスミ**: 絶対に許しません、ゲヘナ――！
