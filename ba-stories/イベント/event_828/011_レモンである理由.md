# 第7話;レモンである理由

*（地図に記された場所を全て調査し、再び海辺に戻ってきた）*

**ウイ**: んん……

**ウイ**: …………あれ？

> **先生（選択肢）**: ウイ？ / 何か引っかかることでもあった？

**ウイ**: そしたら、何か違和感がありまして。

**ウイ**: ……ひとつずつ、説明します。

*（ウイ）まず、ここ……おそらく訓練場として使われていた、迷路のような遺跡。*

*（ウイ）次に調査したのは、ここの地下拠点ですね。*

**ウイ**: そして、残るはレモン農園と灯台。

**ウイ**: 作りは古かったですが、間違いなくトリニティ様式でしたし……

**コハル**: ね、ねぇ……ハナコ……？

**コハル**: あの人……ここに着いた時と雰囲気がちょっと違うような……？

**ハナコ**: ……そうですね。

**ヒナタ**: ウイさん、夢中になると……まるで普段とは別人です。

**ハナコ**: ふふっ、あれが本来のウイさんなのかもしれませんね。

**ウイ**: 調査する前は、拷問の悲鳴が世間に知られないよう、人目を憚っ（はばか）て聖徒会が用意した牢獄や処刑場かと思っていましたが……

**ウイ**: 残念ながら、私の予想は外れてしまったようです。

**ヒナタ**: ……は、ははっ、はは……。

**ウイ**: ……あ。す、すみません……。

**ウイ**: 怖がらせたかったわけではなくて、余計な心配だったな、と。

**ウイ**: ですが、これでは拍子抜けのような……？

**ウイ**: そう思ったら、その……

> **先生（選択肢）**: まだどこか、腑に落ちないんだね。 / じゃあ、もう少し調査してみようか。

**ウイ**: まあ、調べても何も出ないかもしれません。
むしろ、その可能性の方が高いですが……

**ウイ**: 「もしも」の可能性を探ってこそ、発見できる事もあるので。
調べると決めた以上、最後まで諦めるわけにはいかないです。

**ウイ**: 「これで十分」と簡単に線引きしてしまうようでは、研究者としては何も成し遂げられませんから。

**ウイ**: そこに何も見つからなかったとしても、必ず何かあるはず、と視点を変えてみるのが大切です。諦めない気持ちを忘れてはいけません。

**ウイ**: ……今度は、視点を変えてみましょうか。
「この場所の役割」から、

**ウイ**: 「閉鎖された理由」に。

**ハナコ**: そうですね。伝統を重んじ記録を怠らない彼女たちが、何故この島を捨て、記録からも消し去ったのか。

**ハナコ**: まだ私たちの知らない情報があるのかもしれません。

**ウイ**: 同じことを、考えました。聖徒会らしくないな、と……

**ウイ**: ですが、その理由とは……？

**ウイ**: 訓練場、地下拠点、灯台……ここまでは、良いでしょう。

**ウイ**: ……ただ、どうしてこの島にはレモンが？

**ウイ**: ……もしかして。

**ヒナタ**: ウイさん……？

**ウイ**: ……この地図を見てください。
レモン農園だけ、位置が浮いていませんか？

**ハナコ**: 他にも収穫しやすい場所はあったはずなのに、不自然ですね。

**ハナコ**: 島内では、ここでしかレモンの木を見ませんでしたし、おそらく外部から持ち込まれたものでしょう。

**ウイ**: 島で長期滞在するにしても、主食にはなり得ないレモンを……
きっと、それだけの理由があったはず。

**ハナコ**: ……レモン……レモン……。

*[ns5] "ウイ……？"
[ns6] "き、急にバーナーを取り出してどうしたの？"*

**ウイ**: うーん……やっぱり、アレしか考えられませんね。

**ウイ**: これで、よし……。

*[ns7] "う、ウイ！？"
[ns8] "火傷はない……！？"*

**ハナコ**: あら……大胆ですね♡

**ハナコ**: 大人しそうな方ほど、実は……

**ウイ**: ……私の見立ては正しかったようですね。レモン汁で書かれた部分は、酸の影響で水分の保持能力が低くなるので、炙（あぶ）ると先に焦げて浮かび上がります。

**ウイ**: ……いや、もしかしたら、聖徒会はそこまで考えて……。

**ウイ**: とにかく、次の目的地が決まりましたね。
