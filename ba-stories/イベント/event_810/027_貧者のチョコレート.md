# 貧者のチョコレート

**【旧校舎付近の廃墟】**

**ノドカ**: やってしまいました……。

**ノドカ**: 今日はバレンタインデー、どうにか先生に素敵なチョコレートをプレゼントしたかったのですが……。

**ノドカ**: 旧校舎ではチョコレートはおろか、チョコレートの原材料も中々手に入らなくて……頑張ったのですが……

**ノドカ**: これを……。

> **先生（選択肢）**: ……カカオ？

**ノドカ**: はい。実は給食部にお友達がいまして。

**ノドカ**: それで「チェリョンカ」を作るためのカカオの実を、特別に譲ってもらったんです。

**ノドカ**: しかし、これをどうすれば良いのかも見当つかず……。

**ノドカ**: や、やっぱりガッカリされましたよね……すみません、ちゃんとしたものが用意できなくって……。

> **先生（選択肢）**: カカオの実を齧る。

**ノドカ**: せ、先生！？それ、あまり生で食べるようなものではありませんよ！？

**ノドカ**: せめて、お湯で温めるか何か……。

> **先生（選択肢）**: ノドカの気持ちがこもったプレゼントだから、大丈夫。

**ノドカ**: 私の、気持ち……。

**ノドカ**: 先生はこれも、ちゃんとプレゼントとして受け取ってくれるんですね。

**ノドカ**: 私の好きな人は、こういう……。

**ノドカ**: 先生。私、決めました。

**ノドカ**: 今年は失敗してしまいましたが……来年こそは、必ず。カカオの実だけではなく、バターと砂糖も手に入れて、ちゃんとしたチョコレートをプレゼントしてみせますね！

> **先生（選択肢）**: 原材料ではなく、チョコレートをっていう線は……？

*（その後カカオの実のお返しとして、味の参考も兼ねてノドカに美味しそうなチョコレートをプレゼントした。）*

*（それなりに高級だったこともあってか、どうやら半分も食べられなかったらしい……。）*
