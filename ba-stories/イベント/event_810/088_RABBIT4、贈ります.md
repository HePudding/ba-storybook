# RABBIT4、贈ります

**【シャーレ・執務室】**

**ミユ**: あ、先生……。

**ミユ**: お、おはようございます……。

> **先生（選択肢）**: おはよう、ミユ。どうかした？

**ミユ**: その……今日はバレンタインデーと聞きまして……

**ミユ**: いつもお世話になっている先生に、ささやかなお礼をと思い……

**ミユ**: ど、どうぞ……つまらないものですが……。

*（先生（心の声））"ありがとう！これは……？"*

**ミユ**: チョコレートバークといいます……。

**ミユ**: 本当はもう少し豪華なチョコレートを用意したかったのですが……
その、状況的に難しくて……

**ミユ**: 補給品のチョコレートと、採取した果物で作ってみました……。

**ミユ**: 綺麗な形ではありませんが……
私はこういうものしか用意できなくて……。

**ミユ**: うぅ……やっぱり期待外れでしたよね……？

> **先生（選択肢）**: そんなことないよ。ミユの気持ちがこもってて嬉しい。

**ミユ**: ですが……先生は他の方々からもすでにたくさん頂いているでしょうし……

**ミユ**: こんなみっともないチョコレートなんて、
すぐに記憶から消え去ってしまうと思います……。

**ミユ**: そうして私がバレンタインデーにプレゼントを
贈ったことも、忘れられていくんでしょうね……。

**ミユ**: 私は誰の記憶にも残らないまま、落ち葉のように捨てられて……

*（先生（心の声））ミユのチョコレートを食べる。*

**ミユ**: 先生……？

*（先生（心の声））ミユのチョコレートを全力で食べる！*

**ミユ**: せ、先生……！？

**ミユ**: ちょ、ちょっと待ってください……突然どうしたんですか……！？

**ミユ**: そ、そんなに急いで食べたら――

> **先生（選択肢）**: ……けほっ、けほっ！

**ミユ**: だ、大丈夫ですか……？

**ミユ**: 一気にたくさん食べると、喉に詰まらせてしまいますよ……。

**ミユ**: 一体どうしてこんなことを……？

> **先生（選択肢）**: ミユのチョコレートがすごく美味しくて……。 / ……つい我を忘れちゃった。

**ミユ**: ……えっと、でもそれは補給品のチョコレートを溶かして固めただけで……

**ミユ**: ……えっと、でもそれは補給品のチョコレートを溶かして固めただけで……

**ミユ**: 中に入れたのも、その辺で手に入るような果物ですよ……？

*（先生（心の声））"それでもすごく美味しいよ。絶対に忘れたりしない。"*

**ミユ**: ……やっぱり、先生は優しいですね。

**ミユ**: でも私……記憶に残らないのも怖いのですが……

**ミユ**: ……先生が危ない目にあう方が、もっと嫌です。

**ミユ**: だから……

**ミユ**: チョコレートは少しずつ……大事に召し上がっていただけると嬉しいです。
