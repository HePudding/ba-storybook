# 包まずにありのまま

**セナ**: ……ああ、先生。こんばんは。

**セナ**: 深夜にお呼びしてしまってすみません。一応待機中の身でしたので、遠くまで動くわけにはいかず。

> **先生（選択肢）**: ううん、大丈夫。どうしたの？ / 今日もお疲れさま。ところで今日はどんな用事で？

**セナ**: ありがとうございます。

**セナ**: まあいつも通りということもありますし、本日は今のところさほど忙しくはないのですが……。

**セナ**: 先生もいつも通り、いえ今日は特にお忙しいと思いますので、さっそく本題に入りましょう。

**セナ**: さあ、こちらを。

*[ns3] "これ、バレンタインの……？"
[ns4] "包帯が……。"*

**セナ**: はい、バレンタインチョコレートです。後悔はしたくないので、渡さねばと。

**セナ**: はい、バレンタインチョコレートです。後悔はしたくないので、渡さねばと。

**セナ**: 本来ならもっときちんとしたものを、と思ったのですが……。

**セナ**: 残念ながらこのところずっとお店に行く余裕も無く、手元にあった材料でどうにか。

*（先生（心の声））"そうなんだ、ありがとね。"*

*（先生（心の声））"……「手元にあった材料」？"*

**セナ**: はい。ご存知の通り、救急車にはありとあらゆる薬品やら材料がありますから。このようなチョコレートぐらいでしたら、何とか作ることが可能です。

**セナ**: ……？

**セナ**: どうかされましたか？不安そうなお顔ですが。

**セナ**: ……ああ、およそ見当はつきました。ですがご心配なく。

**セナ**: 確かにそれは「チョコレートの味がする化合物」です。しかし身体に害はありません、むしろ良いでしょう。

**セナ**: どちらかというと、その「包装」の方が気になるかもしれませんが…

**セナ**: ちょうど良い物が無かったので、とにかく清潔さが大事かと思いまして。

**セナ**: よく考えるとこういったものはビジュアルも大切なので、今になって実は減点対象だったのではと心配です。

> **先生（選択肢）**: ううん、セナらしくて良いと思う。 / 色々試行錯誤してくれたんだね、嬉しい。

**セナ**: ……そういうところで包み隠さないのは相変わらずですね、先生。

**セナ**: ……そういうところで包み隠さないのは相変わらずですね、先生。

**セナ**: 何にせよ喜んでいただけたようで、ホッとしました。よろしければ今すぐにでもいかがですか？

**セナ**: こういったものは、渡した方から相手に食べさせてあげることも多いと聞きました。せっかくですし実施してみましょう。

**セナ**: ……どうしましたか？何を躊躇うのです？

**セナ**: 心配しないでください、安全ですし清潔です。動かないでください。

**セナ**: ……やむを得ません、強行しますか。

*（そうしてセナから強制的にあーん（？）をされ……確かに味はすごく美味しいチョコレートを味わった）*
