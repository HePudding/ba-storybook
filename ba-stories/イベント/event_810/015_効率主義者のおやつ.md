# 効率主義者のおやつ

**【ミレニアムサイエンススクール】**

*（ミレニアムの校内で、ひとりで何かを食べているエイミを見かけた。）*

> **先生（選択肢）**: おはよう、エイミ。

**エイミ**: あ、おはよう先生。

**エイミ**: ちょうど仕事を終えて帰り道だけど……偶然だね。

> **先生（選択肢）**: 何を食べてるの？

**エイミ**: チョコレート。

**エイミ**: 任務中は結構カロリーを消費するから……任務が終わった後は、糖分がいっぱい入ったチョコレートを食べるのが効率的なの。

**エイミ**: 先生も一つ食べる？

*（先生（心の声））"このチョコレートはいったい……？"*

**エイミ**: どうしたの？

**エイミ**: そういえば甘いものが好きな子も、なぜかこのチョコレートだけは避けてるんだよね……。

**エイミ**: だからなのか売店に行ったら、このチョコレートだけすごい割引されてるのに、たくさん積まれてて……。

> **先生（選択肢）**: その、形がちょっと……。

**エイミ**: チョコレートを食べるのに、形が問題なの？

**エイミ**: どんな形であれ、中の成分は基本的に同じだよ？

**エイミ**: むしろ割引されてるなら、そっちの方を選ぶのは効率的な選択。

**エイミ**: はい、先生。このチョコレートたくさん買ったから、先生にもおすそ分けするね。

*（その後エイミにもらったチョコレートを持ってシャーレに帰る途中、ヴァルキューレの子に誤解されて質問攻めにあった……。）*
