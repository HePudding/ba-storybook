# 後日談その6;ある種のエクストリームスポーツ

**ヒフミ**: ビーチバレー……つまり、二人一組でチームを組み、砂浜で行う形のバレーボールです。

**ヒフミ**: ……なのですが、トリニティの体育の授業では「6人制のバレーボール」しかやったことがないので、ビーチバレーの動きやルールはちょっと良く分からないんですよね……。

**ヒフミ**: だからといって、6人制にするには人数が足りませんし……。

**ヒフミ**: というわけで、今回は特別にスケバンのみなさんに、お相手をお願いすることにしました！

**チンピラ**: あんたら正気か……？あそこまでドンパチやっておいて、ボール遊びに付き合えって？

**チンピラ**: いくら負けた身とはいえ、こっちにだってプライドがあんだよ！

**チンピラ**: あたしらがそんな「はいそうですか」って言うわけが――

*……ありまーす！やりたいです！ぜひやらせてください！*

**チンピラ**: じ、実は結構前から、ビーチバレーに興味あったんだよなぁ～。

**チンピラ**: みんなでやるビーチバレー楽しみだなー！うぇ～い！！！

**ヒフミ**: こんなに快く応じてくださるなんて……。

**ヒフミ**: ありがとうございます！

**ヒフミ**: では、早速始めましょうか？

**ヒフミ**: さて、ネットも設置できましたし……

**ヒフミ**: それでは、ツルギさんのサーブからお願いします！

**チンピラ**: ちっ、泣く子も黙ると有名なあたしらスケバンが、どうしてこんなことに……。

**チンピラ**: はぁ……ま、所詮ボール遊びだ。難しいことでもないし、てきとーに付き合って解散ってことにしよう。

**チンピラ**: あ、そっちにサーブ来――

*（チンピラC）ぐぼぁぁっっ！？*

**チンピラ**: なっ！？

**チンピラ**: な、何だ？空襲か！？

**チンピラ**: い、いや、ただボールが飛んできただけ……の、はず。

**ヒフミ**: つ、ツルギさん。次のサーブをお願いします……できれば、もうちょっと緩めに……。

**チンピラ**: ぐああぁっっ！

**チンピラ**: だ、大丈夫か！？衛生兵！衛生兵ーー！！

**チンピラ**: こ、こんなのビーチバレーじゃない……。

**チンピラ**: 一方的な暴力だ！

**チンピラ**: 暴力反対！どうか平和的な解決を……っ！

*（その後、ツルギのサーブはスケバンが全員行動不能になるまで続いた。）*
