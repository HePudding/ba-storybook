# 第10話;工務部の提案

**【レッドウィンター連邦学園・雪山】**

**モミジ**: はあ、はあ……。

**メル**: 勝った……の？

**モミジ**: 事務局は撃退しましたけど、これが理由で恨まれないか心配です……

**メル**: 大丈夫。「知識解放戦線」は元々事務局に恨まれてるから。

**モミジ**: それはそうですけど……。

*（先生（心の声））"同人の行く手を阻む者には、破滅が待っているだろう……！"*

**モミジ**: せ、先生？

**メル**: とりあえず行くよ！時間がない！

**ミノリ**: ――労働と革命について、話したいことがある。

**モミジ**: 今度は誰！？

**メル**: えっと……？工務部員の方々と、ミノリ部長……？

**ミノリ**: ここにいたんだな、会いに行く手間が省けた。

**ミノリ**: 『ルナ』という作品の関連書籍の制作を、「知識解放戦線」で大々的に進めていると聞いて訪ねてきた。

**モミジ**: えっと……。大々的というほどではないけど、まあ、進めてはいるよ……？

**モミジ**: その……私たちの同人誌と工務部に、何の関係があるのでしょうか？

**ミノリ**: まず、あたしたちは過去の遺産を振り返り、それを現代に蘇らせようという「知識解放戦線」の発想と精神を尊重してる。

**ミノリ**: 労働の歴史を振り返っていくほど、革命への闘志に溢れ、既得権益に対する闘争意欲は、より一層高められるものだからな。

**モミジ**: （『ルナ』ってそんな内容だったっけ……？）

**モミジ**: （絶対に違いますよね……？）

**ミノリ**: よって！あたしたち工務部は「知識解放戦線」に提案がある！

**ミノリ**: 「知識解放戦線」が現在作っている書籍の内容を、より革命的な内容に変えることを提案する!

**工務部の部員**: 提案する！提案する！

**ミノリ**: さらに、労働階級の啓蒙と解放のために、資本家に苦しむ労働者と彼女たちの闘争を主題として取り上げるよう申し入れる次第だ！

**工務部の部員**: 申し入れる！申し入れる！

*（先生（心の声））"一体どこでその話を聞いたの……。"*

**メル**: 一体何を言い出すのかと思ったら……

**モミジ**: 工務部員の方々にはお世話になっていますし、
図書館を訪ねてくださる方もいらっしゃるので感謝の気持ちはありますが……。

**モミジ**: 今回はダメです！

**メル**: 時間がないから無理！

**ミノリ**: ……失望したよ「知識解放戦線」。
普段、厳選してくれた文学作品に多くの助けてもらっているから……

**ミノリ**: 今度こそ一緒に手を組み「ヨーグルトのふたを舐めずに捨てる憎らしいひげチビ」に痛い目を見せられると期待していたのに！

> **先生（選択肢）**: やっぱりそれ、工務部だったんだ！？

**ミノリ**: 仕方がない。この方法だけは使いたくなかったが……。

**ミノリ**: 同志たちよ！これからあたし達は、ぼやけてしまった「知識解放戦線」の視野を、再び明瞭にすべく、熱き闘争へと移行する！

**工務部の部員**: わああああ！
