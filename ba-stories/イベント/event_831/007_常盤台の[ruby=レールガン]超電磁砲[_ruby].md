# 第5話;常盤台の超電磁砲（レールガン）

**【ミレニアム廃墟地下】**

**미사카**: かなり下まで来たはずだけど……

**미사카**: 何か変じゃない？人の痕跡がないのよね。

**미사카**: ……これで本当に道は合ってるの？

**エイミ**: 問題ないよ。それに、痕跡がないのは予想通り。

**エイミ**: 長い間、誰の目にも触れていない証拠だから。

**사텐**: ……あっ、冷凍庫に入れっぱなしのサバ缶、帰ったら何に使おうかな。

**사텐**: って、どうかしましたか、先生？

**사텐**: ……え？

**사텐**: 戦闘になっても驚かない理由、ですか？

**사텐**: うーん……それは、まあ……

**쇼쿠호**: 色々とあるのよ、私たちの世界にも。

**쇼쿠호**: 誰とは言わないけど、よく問題を起こしてる人もいるしぃ？

**사텐**: 御坂（みさか）さんは、学園都市で七人しか存在しない「超能力者（レベル5）」で、「常盤台の超電磁砲（レールガン）」って呼ばれてるんですよ！

**コトリ**: ……むむ？今、なんと言いましたか……？

> **先生（選択肢）**: 超電磁砲（レールガン）……？ / 超能力者（レベル5）……？

**미사카**: 気にしなくていいわよ。ここじゃ満足に能力使えないし。

**미사카**: それより食蜂、（しょくほう）人を何だと思ってるの？

**쇼쿠호**: あらぁ、何のことかしら？

**미사카**: しらばっくれるんじゃないわよ！
少しは悪びれたりするもんでしょうが！

**사텐**: まあまあ、お二人とも。
皆さんが見てますし、ここで喧嘩はちょっと……

**사텐**: ……ん？

**사텐**: あれ、いま何か機械の動く音がしませんでした……？

**사텐**: ……あ、この音です。

> **先生（選択肢）**: もしかして、エイミが言ってた……。 / いよいよ、お出ましだね。
