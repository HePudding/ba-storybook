# こう見えても放課後スイーツ部の一員

*（仕事の途中、息抜きに散歩をすべく外へ出た）*

*（道路の片側にあるバス停で、見慣れた人影が見えた）*

> **先生（選択肢）**: あれ？カズサ？ / こんばんは。

**カズサ**: ……あれ、先生？

**カズサ**: ……あれ、先生？

**カズサ**: どうしたのこんな遅い時間に……もしかして散歩？

**カズサ**: ……ん？私？

**カズサ**: えっと……見ての通りバス待ち中。

**カズサ**: ここから寮に向かうバスは間隔がすごく長くて。
運が悪かったらまるまる20分は待たないといけないの。

**カズサ**: まあ、地下鉄乗ればちょっと早く帰れるけど……。

**カズサ**: ……人混みは苦手で。
ほら、この辺人いないでしょ？バスも同じでガラガラだから。

**カズサ**: 時間が少しかかっても、人が少ないほうが好き。その方が余裕も出るし。

**カズサ**: それに、このバス停には椅子がある。

**カズサ**: つまり……。

**カズサ**: ご覧のとおり、座っておやつを食べる時間もある……ということだよ。ふふ。

**カズサ**: はい、先生もどうぞ。

> **先生（選択肢）**: いいの？ / カズサのジュースでしょ？

**カズサ**: 私の心配はしなくていいよ。他にもあるから大丈夫。

**カズサ**: 私の心配はしなくていいよ。他にもあるから大丈夫。

**カズサ**: ジャーン！こういう風に、マカロンとかも持ち歩いているの。

**カズサ**: ふふっ。こう見えても放課後スイーツ部の一員だよ。
カバンの中のおやつはジュースだけじゃないって事。

**カズサ**: 「スイーツは一人で食べるより、みんなで食べる方がもっとおいしいもの」……。

**カズサ**: ……って、あいつらが言ってたし。ははっ。

**カズサ**: あ、今のはうちのスイーツ部の話。

> **先生（選択肢）**: カズサもスイーツが好きなんだね。 / 部活のそういう雰囲気だけ楽しんでいるのかと思った。

**カズサ**: ん～……やっぱそう見えるのかな。
そりゃ自分でもちょっと不思議だとは思うけど……。

**カズサ**: ん～……やっぱそう見えるのかな。
そりゃ自分でもちょっと不思議だとは思うけど……。

**カズサ**: でもスイーツが嫌いな女子高生って、この世にいると思う？
好きなスイーツの種類が違うだけだと思うな。

**カズサ**: 本当に、他のメンバーにただ付き合うために食べてるわけじゃないからね。
イチゴが乗ったショートケーキも好きだし、マカロンも大好き。

**カズサ**: むしろスイーツが好きすぎて……安物の量産品は避けるぐらいっていうか。

**カズサ**: 話が出たついでに。いつか、先生にも私の「オススメ店」を教えてあげたいな。

**カズサ**: きっと先生も好きだと思う……
あ、まあ……甘いものが好みじゃなかったら、まあまあかもしれない。

**カズサ**: ……うーん、でも意外と大丈夫かも。くどい甘さじゃないし。

**カズサ**: それかもう少しあっさりした方が……。

**カズサ**: ……あ。

**カズサ**: 話してたらバス来ちゃった。もうちょっと後で来ると思ってたのに……

**カズサ**: これに乗らないと、また20分くらい待たないといけないし……

**カズサ**: ……仕方ない。まだおやつもジュースも結構残ってるけど、行かなきゃ。

**カズサ**: んー……。

**カズサ**: 先生、その……一応、念のため。本当に何の他意もないんだけど。

**カズサ**: えっと……その。

> **先生（選択肢）**: 今日の仕事は全部終わってるから。 / 20分ぐらいは大丈夫だよ。

**カズサ**: ……はぁ。

**カズサ**: 本当に……先生は……

**カズサ**: ……いつも思うけど、他の子にもそんな風に言ってるの？

**カズサ**: いつか誰かが先生に襲いかかったとしても、その時は自業自得だからね。
肝に銘じておきなよ、分かった？

**カズサ**: うーん……でも、まあ……。

**カズサ**: もし本当にそんな事が起きたら……。
連絡してきてもいいよ。授業時間じゃなければ助けてあげるから。

**カズサ**: ……ん？

**カズサ**: 冗談に聞こえた？

**カズサ**: はぁ、またそんな風に……

**カズサ**: ま、いいよ。今はそういう事にしといてあげる。
