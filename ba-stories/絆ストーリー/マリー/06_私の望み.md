# 私の望み

*（マリーに指定された場所は、トリニティのとある射撃場だった）*

*（広い空間でほとんど音も無く、下手をすれば大聖堂や図書館よりも静まり返っていた）*

*（マリー）……お越しいただいてありがとうございます、先生。*

**マリー**: 急なご連絡になってしまって申し訳ありません。

**マリー**: その、本当に大したことではなかったのですが……。

> **先生（選択肢）**: 気にしないで。ところで、ここは……。 / マリーが射撃場……何だかちょっと見慣れないかも？

**マリー**: あ、ここはですね、その……。

**マリー**: あ、ここはですね、その……。

**マリー**: 特にお話したい内容が射撃や銃に関係しているわけではないのですが……ただ、静かな場所ですので。

**マリー**: ご覧の通り普段は誰もおらず、結構静かでして。お祈りにも集中しやすい場所なんです。

**マリー**: タイミングによっては大聖堂や、懺悔室の中よりも静まっているかもしれません。

**マリー**: ですので時々、何かに悩んだときやお祈りをする時などに、ここに来ることがあるんです。

**マリー**: も、もしかして変、でしょうか……？

> **先生（選択肢）**: ううん、そんなことないと思うよ。 / そう聞いてみると納得したし、何となく似合ってるかも。

**マリー**: あ、ありがとうございます。

**マリー**: あ、ありがとうございます。

**マリー**: 実は私も、元々この場所は全然知らなくって……。

**マリー**: ただ、時々「他のシスターの方々も訪れる場所」とは聞いていたのですが……。

**マリー**: ですが実際にこうして来てみると、この静かな空気は……。

**マリー**: 心を落ち着かせたい時や、自らのことを反省する時などにぴったりの空間なんです。

**マリー**: それ以外にも例えば、自らを励ましたい時や、雑念を無くしたい時……

**マリー**: 雑念……

*（先生（心の声））"……マリー？"*

**マリー**: そ、その……。

**マリー**: 実は、以前お休みを頂いた時にも、ずっと自分なりに悩んだりしていたのですが……。

**マリー**: 他のシスターにも聞いてみたところ、「誰にでも時折そういうことはある」と言われまして……。

**マリー**: それでも私は、悩んでしまって……これで良いのかな、と。自己管理ができていないのでは、と……。

**マリー**: ですが、それに対しても「奉仕する気持ちは大切だけれど、自分に対して奉仕してあげる気持ちも時には大切だ」、とのことでして……。

**マリー**: それで、考えてみたんです。

**マリー**: 今日も先ほどまで、ここでお祈りをしながら……それで、辿り着きました。

**マリー**: ですがそれは自分でも、あまりに予想外だったと言いますか、どうして良いのか迷うと言いますか……。

**マリー**: ……に。

> **先生（選択肢）**: ……に？ / ……？

**マリー**: 先生と、一緒に……。

**マリー**: 先生と、一緒に……。

**マリー**: その……時々、本当に時々で良いので……。

**マリー**: あの時のように、私と、その……。

**マリー**: 一緒に……。

**マリー**: 一緒の、時間を……過ごしたいな、と……その、特別何かをしなくても、私は大丈夫ですし……。

**マリー**: 以前来ていただいた時……私、何と言えば良いのでしょうか……本当に、嬉しかったんです。

**マリー**: も、もちろん。自分でも分かってはいるのです。こういった感情は、シスターらしくないものだと。

**マリー**: 面倒を見てほしい、付き合ってほしい、すぐそばにいたい……。

**マリー**: こういった気持ちは、誰かのために行動するべきシスターとしての道とは、下手をしたら相反するようなものにも思えます。

**マリー**: 分かっては、いるのですが……。

**マリー**: ……？

**マリー**: 喜んで……？

**マリー**: そ、そんなあっさり……！

**マリー**: 良いの、ですか……？

> **先生（選択肢）**: もちろん。それに、生徒と先生なんだから。 / 24時間、ずっとシスターじゃなくても良いんじゃないかな？

**マリー**: ……！

**マリー**: ……！

**マリー**: 先生……。

**マリー**: ……ふふっ。少し、考え過ぎていたのかもしれません。

**マリー**: はい、それでは……立派なシスターになりたいという気持ちも、もちろん今まで通り持ちつつ……。

**マリー**: これから私も、ひとりの生徒として。あらためて先生と接していきたいと思います。

**マリー**: 思えば、先生に対しては「話す」ということを……当の私が忘れてしまっていましたね。いつも、「ぜひ話してください」と何度も口にしていたのに。

**マリー**: こうしてお話をして気持ちを共有できただけで、すごく楽になりました。

**マリー**: ……本当に、ありがとうございます。

**マリー**: それではこれからも、

**マリー**: よろしくお願いします。先生。

**マリー**: ……甘えたがりの私のことも、許してくださいね？

*（その後マリーと一緒にお互いお祈りをしてから、シャーレへと戻った）*
