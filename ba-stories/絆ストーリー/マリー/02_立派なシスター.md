# 立派なシスター

**【トリニティ大聖堂の裏側】**

*（マリーに案内してもらった通り、トリニティ大聖堂の裏手へと向かった）*

*（？？？）……あ。こちらです、先生。*

**マリー**: お待ちしておりました。

**マリー**: 突然のことでしたが、来ていただいてありがとうございます。

**マリー**: ……え？

**マリー**: この時間帯、ですか？いえ、特に何かがあるわけではないのですが……。

**マリー**: すみません、私が空いていたのがこの時間しかなく……。

> **先生（選択肢）**: そっか、忙しいところに来ちゃってごめんね。 / もっと時間のある時に来れば良かったね。

**マリー**: ああいえ、そんな、大丈夫です。お越しいただけて嬉しいですから。

**マリー**: ああいえ、そんな、大丈夫です。お越しいただけて嬉しいですから。

**マリー**: それに、いつも通り大聖堂でのお仕事をしていただけですし。

**マリー**: ……お仕事の内容、ですか？

**マリー**: そうですね、

**マリー**: もちろんシスターとしてやるべきことは多岐にわたるのですが……。

**マリー**: 主に私にできることと言えば、懺悔などお話をお聞きし、心の中の痛みを共有させていただくことくらいです。

**マリー**: 大したことはできないのですが、それでもそういった時間を通じて、何とかご自分で立ち直ることができるという方もいらっしゃいまして。

**マリー**: ……アドバイザー、ですか？

**マリー**: た、確かに、そういった面も一部あるかもしれませんね……。

**マリー**: ですが、私にはそんな、アドバイスなんて大それたことはできなくって……私にできることは、本当にただお聞きすることだけです。

**マリー**: 私にはそれくらいしかできませんが……お話を聞くことで、その方が少しでも前に進めるのであれば。

**マリー**: 私はそれを、心から嬉しく思うのです。

**マリー**: というわけで……

**マリー**: こほん。先生はどんなお悩みをお持ちなのか、よろしければお聞かせいただけますか？

**マリー**: ……？

**マリー**: あ、本日は特にお悩みがあって、というわけではないのですが……？ただ、ご挨拶を……？

**マリー**: そ、それは失礼しました……。

**マリー**: 何だか調子に乗って勘違いしてしまいましたね……？お恥ずかしい限りです……。

**マリー**: え、普段から頑張ってるのがよく伝わってきて、もし悩みがあったとしてもすでに無くなってそう……？

**マリー**: ……ふふっ。

**マリー**: ありがとうございます、先生。

**マリー**: そのようなお言葉を頂けて、何だかとっても嬉しくなりました。

> **先生（選択肢）**: マリー自身は大丈夫？無理してない？ / マリーも辛くならないように、しっかり休んでね。

**マリー**: ……いえ。

**マリー**: ……いえ。

**マリー**: そう言っていただけるのは嬉しいのですが、甘えるわけにはいきません。私にはまだまだやるべきことが多く、往くべき場所も遠いので。

**マリー**: 今はこうして、いっぱいいっぱいの私ですが……。

**マリー**: いつかは立派な「シスター」になりたいのです。

> **先生（選択肢）**: シスターに「なる」？ / 今はまだシスターではないの？

**マリー**: うーん……何と言えば良いのでしょう……。

**マリー**: うーん……何と言えば良いのでしょう……。

**マリー**: 例えばシスターの服を着たからと言って、それだけで誰もがすぐにシスターになれるわけではない……そういった感じでしょうか？

**マリー**: まだまだ学ばねばならないことも、知らなければならないことも多いですし……。

**マリー**: それに私は、今でも「シスター」と呼ばれることに少々照れてしまいます。

**マリー**: いつか誰かが「シスターマリー」と呼んでくださっても、その時恥ずかしがることなく応えられるように……それが今の、私の夢なんです。

**マリー**: ですから今はまずこうして、できることを一つ一つ全力でやらないと。

**マリー**: ……？

**マリー**: すごく立派……ですか？

**マリー**: 心意気はすでに、シスターそのもの……？

**マリー**: か、からかわないでください……！わ、私がまだまだ未熟者なのは、私自身が誰よりも分かっております……！

**マリー**: も、もちろん嬉しくはありますが……今のは聞かなかったことにします。もう、先生ったら……。

> **先生（選択肢）**: でも、本当にそう思うよ。 / シスターマリー！シスターマリー！

**マリー**: も、もう……！

**マリー**: も、もう……！

**マリー**: や、やめてくださ――

*（マリー）……！？*

**マリー**: せ、先生！？
