# 第2話;ポストモーテム（２）

**アズサ**: ……セイアはこう言った。

**アズサ**: エデン条約によって、ゲヘナとトリニティ間の紛争が無くなれば、アリウスの問題はじきに解決できるかもしれない。

**アズサ**: この両校の根深い問題を解決する方法は、少なくとも今はこのエデン条約しかないだろう……と。

**アズサ**: ……しかし同時に、アリウスはティーパーティーのヘイローを破壊しようとしていた。それが実行されれば、キヴォトスは本物の戦場になってしまうはず。

**アズサ**: それを止めたいかどうか……そう聞かれたんだ。

**セイア**: ……そうか。

**セイア**: ……先に言っておこう、白洲アズサ。私は君の考えには首肯しない。

**セイア**: なにせ私は、5番目の古則……つまり「楽園の証明」なんて不可能だということを、誰よりもよく知っているからさ。

**セイア**: だが……

**セイア**: 全てが破局へ向かうのだとして。それを知って尚、君が足掻くと言うならば……

**セイア**: 君に、私の知恵を貸そう。

**セイア**: それがどんな結果になるかは、何も保証できないがね――

*……そして、セイアさんの部屋を爆破させた？*

**アズサ**: ……「セイアが死んだ」と偽装するためには必要なことだった。アリウススクワッドをも騙し切るための手段は、限られていたし……。

**アズサ**: セイアは全ての事件を隠蔽するための助っ人として、救護騎士団の団長を指名した。この後は全て彼女に託せ、と。

**アズサ**: ……そして部屋を爆破して逃げた。その後のトリニティでの騒動や情報を総合して、スクワッドは私が任務に成功したのだと信じた。

**アズサ**: ……これが事の顛末だ。

**ハナコ**: ……。

**アズサ**: その後については、私も知らなかった。まだ意識を取り戻せていないということも……。

**ハナコ**: セイアちゃんの外傷は、すでに完治しているそうです。セイアちゃんの意識が戻らないのは、それ以外の理由みたいで……。

*（ティーパーティーの監察官）ではあなたは、襲撃命令を下したのがミカ様だとは知らなかったと？*

**アズサ**: それは……

**アズサ**: ミカさんは自身の正体をずっと隠しきっていました。アズサちゃんのような命令系統において一番下の位置からは、知る由もなかったでしょう。

**アズサ**: あ、アズサちゃんは人殺しではありません……！あの時もアズサちゃんは、ナギサ様を救おうとして……！

*（ティーパーティーの監察官）……すみません。私はこうして話を聞くことが仕事ですので。*

**ハナコ**: ……ハナコ、ヒフミ。私は大丈夫。

*私に実際に命令を下したのは、アリウススクワッドのリーダー「サオリ」。スクワッドの指揮を執ってるのは彼女だ。*

**サクラコ**: ……そうですね。

**サクラコ**: とにかく、白洲アズサさん……彼女がトリニティに転校してきて、そして実際にナギサさんを守り抜いた。そのことは明白です。

**サクラコ**: その過程で様々なことがあったとはいえ、特別学力試験にも合格。補習授業部は彼女を含めて全員、明確な結果を残しています。

**サクラコ**: 文句の付けようなど無いでしょう……彼女の書類は、私が正式な物にしておきます。

**サクラコ**: シスターフッドが保証しましょう。誰にも、異議申し立てなどさせません。

**ハナコ**: ……ありがとうございます、サクラコさん！

*（先生（心の声））"本当にありがとう。"*

**サクラコ**: ……これで白洲アズサさんは、正式にトリニティの生徒となりました。

**サクラコ**: まだ問題は山積みですし、特にアリウス分校のことは何も解決できていませんが……。

**サクラコ**: 取り急ぎ、目の前の問題だけは落ち着いたと言えるかもしれませんね。

**サクラコ**: お疲れ様でした。ハナコさん、そして[USERNAME]先生。
